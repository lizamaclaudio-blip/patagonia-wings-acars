Patagonia Wings ACARS - paquete completo de perfiles técnicos

Incluye perfiles JSON locales para los addons cargados/definidos en esta actualización.
Pendientes NO incluidos: B350_BLACKSQUARE, BE58_BLACKSQUARE_PRO, TBM9_BLACKSQUARE.

Instalación:
1) Copiar la carpeta PatagoniaWings.Acars.Master sobre la raíz del proyecto ACARS.
2) Reemplazar archivos cuando Windows lo solicite.
3) Compilar en Visual Studio.
4) También se incluyen copias en bin/Debug, bin/Release, bin/x64/Debug y bin/x64/Release para pruebas directas.

Regla:
- Web/flota/reservas usa tipo operativo: C208, A320, B789, etc.
- ACARS usa perfil técnico: C208_MSFS, A320_FENIX, B789_HORIZONS, etc.
